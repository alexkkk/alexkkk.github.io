git中tag用法

发表于 2016-11-17

add tag

git tag -a release_1.0.0 -m “add tag release 1.0.0”

push tag to remote

git push origin –tags

delete local tag

git tag -d release_1.0.0

delete remote tag

git push –delete origin release_1.0.0
